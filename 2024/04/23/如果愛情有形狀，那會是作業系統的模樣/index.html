如果愛情有形狀，那會是作業系統的模樣

Duck Cone
2024-04-23
日常故事

凌晨 2:00 ，我還在為了下週的考試啃書， Ｌ 小姐雙手仍啪嗒啪嗒地敲著鍵盤。

「前輩妳還不下班嗎？」我拖著疲憊的大腦問道。

L 小姐邊敲著鍵盤邊說：「實驗還在跑，待會還要把結果繪製成圖，不知道還要忙多久。」

理工人的生活就是如此樸實無華，我看向眼前散落桌面的筆記以及高藍光傷害的螢幕，心裡嘀咕著今晚可能又要在實驗室過夜。

「最近我跟我男友吵了一架。」L 小姐說道。

L 小姐是我的前輩，在工作上時常受到她的照顧。最近她似乎和交往了三年的男友（以下簡稱 M 先生）有些摩擦。

「我覺得我很難察覺到自己真正的狀態，所以會需要一點時間消化梳理自己的情緒，即便有什麼煩惱，也不太會跟其他人說。」L 小姐說。

「那天他來載我，連我自己都沒發現我自己心情不好，可是他卻一直問我怎麼了，為甚麼不說話，但我自己也不知道阿！」

「而且他看我兩三次沒反應之後，他也開始不開心了。」

「我覺得，」我說：「他可能已經把一切都想好了，就像 Dijkstra Algorithm 已經找出最短路徑，所以他想要趕快把問題解決，但他沒考慮到你還正在思考。」

「嗯…」

「簡單來說，他就像一顆高效能的 CPU ，他把這一盤棋的結果都看透了，也有自信可以解決，但前輩你的話…」

「可是我就像單核心的處理器，一次只能處理一件事，而且速度也比較慢！」L 小姐豁然開朗地說道。

「像是前幾天我因為失眠，所以睡不好也吃不好，當我有點食慾時傳了訊息告訴他我感覺有點餓了，但他卻問我為甚麼不告訴他失眠的事，以及為甚麼不好好吃飯。」L 小姐說道：「當下我真的蠻無言的。」

L 小姐眼中透漏出一絲無奈，但對於 M 先生的回應卻也無可奈何。我喝了一口威士忌杯中的綠茶說道：「我認為 M 先生是關心你的，只是不知道要怎麼表達，就像很多父母是為了小孩子好，但說出來的話像是道德綁架一樣。或許 M 先生可以改成說：『那今天早些休息，別讓自己太累了。』會更妥當一些。」

「對！以前的他真的會這樣說！但現在都沒有了，現在連我忙到沒時間好好吃飯他都會生氣！」

「我的想法是，」我思考了一會兒後說：「 M 先生想要『預防』事件的發生，但前輩你想要『解決』已經發生的事件。」

「加上 M 先生已經全部都想完了，然後在等待你，就像 process fork 出一個 child 後， parent process 需要 waiting child process 一樣，但是 waiting 太久了，最後 child process 變成 zombie ，然後他就爆掉了。」

凌晨 2:45 ，我覺得我已經有點神智不清了，連續兩天都住在實驗室果然還是太操勞，L 小姐今天也是留下來連夜加班，果然理工人的歸宿都是無盡的加班地獄。

「不過前輩，我覺得蠻神奇的。」

「嗯？」

「一般來說，通常是男生會自己默默的想，然後女生會比較盧小小，可是感覺，你們的立場反過來欸哈哈哈！」

「欸！好像是欸哈哈哈！」

我與前輩相視而笑。

「唉～為甚麼我一個母胎單身的男子常常在幫人解決這些事情。」難過的淚水流過我四十五度的臉角(無奈)。

「哈哈哈哈！沒事啦，M 先生也是我的初戀。」

「但是前輩，我覺得這也不是壞事。」

「怎麼說？」L 小姐問道。

「感覺你們兩人都是理性感性參半，只是兩人的思考速度跟角度不同而已，他想的比較快比較全面，而妳則是比較慢但比較謹慎，你們只是需要時間再磨合。」

「或許，」我思考了一下後說道：「下次當事件發生的時候，妳可以設定一個 break point ，去 trace 他的想法，畢竟 M 先生就像一個 Linux kernel 一樣，包含很多程式碼與 function ，我們一般人是無法跟上他的速度的，但可以找一個進入點去嘗試理解他思考的模式。」

「天哪這好 OS 哈哈哈！」L 小姐邊大笑邊說道：「所以我要開啟 gdb 偵錯模式嗎？」

「沒錯就是這樣！」（果然是理工人）

L 小姐繼續說道：「其實他給我的建議我都有聽進去，通常我都會回覆『嗯嗯』來表示我了解了，但他看到我這樣的回覆之後，好像會覺得我在敷衍他。」

（沉默…）

「包含上次我玩遊戲，有一個關卡一直過不了，當時他看到後，也給了我一些建議，但我就比較笨阿，大招什麼的總是會慢半拍，最後他就生氣了，好像認為我過不了是因為沒有照著他說的做。」

「哎呀…」我嘆了一口氣後說道：「M 先生感覺是個很聰明的人，上次跟他談過話後，我就覺得他腦袋似乎時時刻刻都在思考，讓我也很好奇他到底都在想些什麼。所以我認為很多事情他都想過了，而且也都找出最佳解，才會因為前輩沒照著他的方式做而讓他有些不高興。」

「嗯…」

「不過前輩妳下次可以試著調整回覆的方式，畢竟文字敘述本來就容易讓人誤會，加上『嗯嗯』這類的語助詞每個人解讀都不同。」

「既然你們在交往，一定會有很多地方需要磨合啦！包含我認為，除了彼此欣賞對方的優點之外，也要能接受對方的缺點，有時候還要承接住對方的情緒，畢竟不是像一般朋友一樣可以選擇要不要接受對方的缺點，既然都在一起了，就必須了解到這點。」

「了解！」

我看向螢幕上的時鐘，已經是凌晨 3:30 了，我將桌面簡單收拾過後，抱著我的筆記躺向實驗室的沙發床。

「今天謝謝你給了我那麼多建議！」

「不會啦！雖然沒有交往經驗的我給了那麼多建議好像也怪怪的哈哈哈！不過人與人的相處就是這樣子嘛。」

我緩緩地閉上眼，用盡最後的力氣回覆 L 小姐，雖然還想再多聊一些，但我的意識逐漸模糊，最終沉沉地睡去。
